# 🤖 Telegram-бот на Node.js

Простой эхо-бот, который отвечает на команду /start и повторяет любое сообщение.

## 🚀 Живой пример

👉 [@xVolkxDemoBot](https://t.me/xVolkxDemoBot)

## 🛠 Технологии

- Node.js
- Telegram Bot API
- dotenv (переменные окружения)

## 📦 Как запустить локально

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/xVOLKx/telegram-bot-demo.git
   cd telegram-bot-demo
   ```
2. Установи зависимости:
    ```bash
    npm install
    ```
3. Создай файл .env и добавь токен:
    ```bash
    BOT_TOKEN=твой_токен_от_BotFather
    ```
4.  Запусти:
    ```bash
    node bot.js
    ```
