## <img src="images/install.svg" width="24"> Как запустить локально

1. Клонируй репозиторий:
    ```bash
    git clone https://github.com/xVOLKx/telegram-bot-demo.git
    cd telegram-bot-demo
    ```
2. Установи зависимости:
    ```bash
    npm install
    ```
3. Создай файл .env и добавь токен:
    ```bash
    BOT_TOKEN=твой_токен_от_BotFather
    ```
4. Запусти:
    ```bash
    node bot.js
    ```
